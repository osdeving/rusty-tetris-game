# Rust Tetris Game


## Installing SDL2 on Ubuntu

```bash
$ sudo apt install libsdl2-dev
```

Or (https://github.com/libsdl-org/SDL/blob/SDL2/INSTALL.txt)

```bash
$ git clone https://github.com/libsdl-org/SDL.git
$ git checkout SDL2
$ ./configure; make; make install
```

## Installing Rust
